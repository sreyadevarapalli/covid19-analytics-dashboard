# 🌍 COVID-19 Analytics Dashboard

A modern, responsive COVID-19 Analytics Dashboard built with **React**, **Vite**, **Tailwind CSS**, and **Recharts**. The application provides global and country-level COVID-19 statistics, interactive visualizations, and analytical insights using real-time public APIs.

# 📦 Installation

Clone the repository

```bash
git clone https://github.com/sreyadevarapalli/covid19-analytics-dashboard.git
```

Go to project

```bash
cd covid19-analytics-dashboard/frontend
```

Install dependencies

```bash
npm install
```

Run project

```bash
npm run dev
```

# 🌐 API Used

Disease.sh API

### Global Statistics

```
https://disease.sh/v3/covid-19/all
```

### Countries

```
https://disease.sh/v3/covid-19/countries
```

### Country Details

```
https://disease.sh/v3/covid-19/countries/{country}
```
