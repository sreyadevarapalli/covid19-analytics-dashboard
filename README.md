# COVID-19 Analytics Dashboard

A React-based COVID-19 analytics dashboard with interactive statistics, charts, country comparison, filters, and country details.

## Features

- Global COVID-19 statistics
- Country-wise data
- Country search and filtering
- Pagination
- Country comparison
- Analytics charts
- Country details
- Responsive design

## Technologies

- React
- Vite
- JavaScript
- Tailwind CSS
- Recharts
